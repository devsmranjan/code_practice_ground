# Pattern 2

## Question

```markdown
1. You are given a number n.
2. You've to create a pattern of \* and separated by tab as shown in output format.
```

## Input Format

```markdown
A number n
```

## Output Format

```markdown
-   -   -   -   -
-   -   -   -
-   -   -
-   -
-
```

## Constraints

```markdown
1 <= n <= 100
```

## Sample Input

```markdown
5
```

## Sample Output

```markdown
-   -   -   -   -
-   -   -   -
-   -   -
-   -
-
```
